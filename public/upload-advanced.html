Advanced settings

Filename of the encrypted file in your GoogleDrive. Name is publicly available, not protected.
Masks encrypted file in the storage - random file name, hides mime type. Hides original file name from the attacker.
File size concealing helps to hide original message/file size by inflating encrypted file with random noise (noise gets removed on download). It is highly recommended to keep this ON.
Embeds encrypted file in the PNG image so people see the file contains encrypted content. With this option encrypted file in your GoogleDrive will look like PNG file. Warning! All image manipulation (e.g. rotation, flip, resizing, ...) on the produced file will wipe out encrypted content.
